# Basis

## Features

- [x] Spash Screen
- [x] Email Screen
- [x] OTP Screen

Download the app in zip or run

```
git clone https://github.com/joysonjson/Basis.git
```

Then, run the following command:

```bash
$ pod install
```

Open Basis.xcworkspace , Build and run the app.
